# React Text Expander

A lightweight React component that expands and collapses long text, with a customizable character limit — perfect for previews of articles, comments, or product descriptions.

## Features

- Toggle between preview and full text
- Customizable character limit via props
- Clean, reusable component design
- Built with modern React hooks

## Prerequisites

Before running this project, make sure you have installed:

- Node.js (v18 or higher)
- npm

## Installation

1. Clone the repository:
   git clone https://github.com/AREF-Esnaashari/TEXT-EXPANDER-

2. Navigate to the project directory:
   cd TEXT-EXPANDER-

3. Install the required packages:
   npm install

4. Run the project in development mode:
   npm run dev

5. Open the following address in your browser:
   http://localhost:5173
